# 📊 Calculadora de Média Escolar

Projeto simples desenvolvido para calcular a **média dos 4 bimestres**, somando as notas e dividindo por 4.

Além da lógica em JavaScript, o foco também foi **praticar e melhorar o uso de CSS**, evoluindo um código antigo.

---

## 🚀 Funcionalidades

* Inserção das 4 notas bimestrais
* Cálculo automático da média
* Exibição do resultado na tela
* Interface simples e organizada

---

## 🧠 Conceitos praticados

* Manipulação do DOM
* Eventos (`click`)
* Funções
* Operações matemáticas
* Estruturação de layout com CSS
* Estilização e organização visual

---

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript

---

## 📌 Objetivo do projeto

* Reforçar conceitos básicos de JavaScript
* Praticar CSS na construção de interfaces
* Melhorar e versionar um projeto antigo
